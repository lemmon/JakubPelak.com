O mne

Pochádzam zo Slovenska a pracujem ako freelancer. Mojou hlavnou náplňou je frontend a backend development. V spolupráci s ďalšími vývojármi a grafikmi, ponúkam aj full-service výrobu webov a webových aplikácií vrátane konzultácii a optimalizácie.

Profesionálne pôsobím v tejto oblasti približne 9 rokov a programovaniu ako takému sa venujem viac ako 17 rokov.

Momentálne som k dispozícii na freelance zakázky.

Služby

  • Programovanie PHP a MySQL (LAMP a MAMP)
  • NoSQL databázy (MongoDB)
  • Grafický design web stránok a User Interface (UI) design
  • Kódovanie HTML šablón, CSS (LESS) a JavaScript (jQuery)
  • Portálové riešenia
  • Webdesign a optimalizácia pre mobilné telefóny a tablety / Responzívny design
  • Konzultácie v oblasti SEO (Search Engine Optimalization) a použiteľnosti
